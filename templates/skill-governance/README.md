# skill governance template

这是一套可复制的 skill 规则模板。

适合场景：

- 你已经有一个能跑的 skill
- 但想把它从“能用”变成“可维护、可迭代、可传承”

复制后重点改这几类内容：

1. `SKILL.md`
2. `references/constitution.md`
3. `references/gotchas.md`
4. `references/output-contract.md`
5. `references/checklists/`
6. `examples/`

如果还要记录设计原因和轻量上下文，再补：

7. `docs/decisions.md`
8. `references/memory.md`
